Mysql-流程控制结构

2018-10-02   Mysql 

流程控制结构

分支结构

特点

  • if函数

功能：实现简单双分支

位置：可以作为表达式放在任何位置

语法

if(条件，值1，值2)
  • case结构

功能：实现多分支

位置：可以放在任何位置，
如果放在begin end 外面，作为表达式结合着其他语句使用
如果放在begin end 里面，一般作为独立的语句使用

语法

语法1

case 表达式或字段
when 值1 then 语句1;
when 值2 then 语句2；
..
else 语句n;
end [case];

语法2

case 
when 条件1 then 语句1;
when 条件2 then 语句2；
..
else 语句n;
end [case];
  • if结构

功能：实现多分支

位置：只能放在begin end中

语法

if 条件1 then 语句1;
elseif 条件2 then 语句2;
...
else 语句n;
end if;

循环结构

位置：只能放在begin end中

特点：都能实现循环结构

对比：

这三种循环都可以省略名称，但如果循环中添加了循环控制语句（leave或iterate）则必须添加名称

loop 一般用于实现简单的死循环

while 先判断后执行

repeat 先执行后判断，无条件至少执行一次

  • while

语法

【名称:】while 循环条件 do
        循环体
end while 【名称】;
  • loop

语法

【名称：】loop
        循环体
end loop 【名称】;
  • repeat
    语法
语法：
【名称:】repeat
        循环体
until 结束条件 
end repeat 【名称】;
  • 循环控制语句

leave：类似于break，用于跳出所在的循环

iterate：类似于continue，用于结束本次循环，继续下一次
