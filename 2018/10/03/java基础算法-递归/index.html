java基础算法-递归算法

2018-10-03   递归算法 

递归思想：从最外部到达最里部，从最里部开始解决问题，直到解决完最外部的问题。

基本含义

是指函数/过程/子程序在运行过程序中直接或间接调用自身而产生的重入现象。

在计算机编程里，递归指的是一个过程：函数不断引用自身，直到引用的对象已知。

使用递归解决问题，思路清晰，代码少。但是在主流高级语言中（如C语言、Pascal语言等）使用递归算法要耗用更多的栈空间，所以在堆栈尺寸受限制时（如嵌入式系统或者内核态编程），应避免采用。所有的递归算法都可以改写成与之等价的非递归算法。

示例

计算n的阶乘

package test;

class Recursion {

    public static void main(String[] args) {

        long result = testRecursion(10);
        System.out.println(result);
    }

    /*
     * 计算n的阶乘
     */
    static int testRecursion(int n) {

        //当n=1时终止递归
        if(n==1) {
            return 1;
        }

        return n * testRecursion(n-1);
    }


}

由此可见，递归算法本质就是方法本身调用自己，每次调用时传入参数不同，递归体内含有方法结束条件，当传递的参数符合结束条件时，最内层的方法结束调用，以此达到迭代的效果。
