java基础算法-快速排序

2018-10-04   快速排序 

解释

快速排序（Quicksort）是对冒泡排序的一种改进。

原理

它的基本思想是：通过一趟排序将要排序的数据分割成独立的两部分，其中一部分的所有数据都比另外一部分的所有数据都要小，然后再按此方法对这两部分数据分别进行快速排序，整个排序过程可以递归进行，以此达到整个数据变成有序序列。

做法

  • 选择一个关键值作为基准值。比基准值小的都在左边序列（一般是无序的），比基准值大的都在右边（一般是无序的）。
  • 从后往前比较，用基准值和最后一个值比较，如果比基准值小的交换位置，如果没有继续比较下一个，直到找到第一个比基准值小的值才交换。找到这个值之后，又从前往后开始比较，如果有比基准值大的，交换位置，如果没有继续比较下一个，直到找到第一个比基准值大的值才交换。直到从前往后的比较索引>从后往前比较的索引，结束第一次循环，此时，对于基准值来说，左右两边就是有序的了。
  • 接着分别比较左右两边的序列，重复上述的循环。

示例

package test;

public class Demo {

    public static void main(String[] args) {

        int[] arr= {5,7,4,6,5,8,6,9};

        //定义数组的起始索引
        int low=0;

        //定义数组的末尾索引
        int high=arr.length-1;

        //测试
        sort(arr,low,high);

        //打印结果
        for(int i:arr) {
            System.out.print(i+"\t");
        }

    }

    static void sort(int [] arr,int low,int high) {

        int start=low;
        int end=high;

        //确定基准值
        int key=arr[start];

        //判断end是否大于start，当等于时，比较结束
        while(end>start) {

            //从后开始比较
            while(end>start&&arr[end]>=key) {
                end--;
            }

            //当后值小于基准值调换二者位置
            if(arr[end]<key) {
                int temp=arr[end];
                arr[end]=arr[start];
                arr[start]=temp;
            }

            //从前开始比较
            while(end>start&&arr[start]<=key) {
                start++;
            }

            //当前值大于基准值时调换二者位置
            if(arr[start]>key) {
                int temp=arr[start];
                arr[start]=arr[end];
                arr[end]=temp;
            }
        }

        //比较结束之后，此时start等于end，即基准值索引,得到左边小于基准值的无序序列，右边大于基准值的无序序列

        //比较low与基准值索引，之间为左无序序列
        if(low<start) {

            //递归进行排序
            sort(arr,low,start-1);
        }

        //比较high与基准值索引，之间为右无序序列
        if(high>end) {

            //递归进行排序
            sort(arr,end+1,high);
        }

    }

}
