java基础学习-内部类的四种实现方式

2018-10-05   内部类 

静态内部类

只能访问外部类的静态成员变量和静态方法，生成静态内部类的方式为

*.Inner inner=new *.Inner();

public class Outer{

    //外部类的静态属性
    static String str="测试";

    public static void main(String[] args) {

        //测试静态内部类的静态方法
        Outer.Inner.test02();

        //生成静态内部类对象
        Outer.Inner inner=new Outer.Inner();

        //测试静态内部类的非静态方法
        inner.test01();

    }

    //定义静态内部类
    static class Inner{

        String name="静态内部类";

        //静态方法
        void test01() {
            System.out.println(str);
        }

        //非静态方法
        static void test02() {
            System.out.println(str);
        }
    }
}

成员内部类

  • 可以当做成员变量来看待理解，可以访问外部类的静态与非静态方法和成员变量，生成成员内部类的方式（例）

Outer o = new Outer（）；

Inner i = o.new Inner（）；

  • 当内部类和外部类有同名的成员时，内部类可通过 外部类名.this.变量名 访问外部类成员。
public class Outer {

    //外部类的静态私有属性
    private static String str01="测试静态";

    //外部类的非静态私有属性
    private String str02="测试非静态";

    public static void main(String[] args) {

        //实例化成员内部类之前需先实例化外部类
        Outer outer=new Outer();

        //实例化成员内部类
        Outer.Inner inner=outer.new Inner();

        inner.test01();
        inner.test02();
        inner.test03();

    }

    class Inner{

        //测试访问外部类的静态属性
        void test01() {
            System.out.println(str01);        
        }

        //测试访问外部类的非静态属性
        void test02() {
            System.out.println(str02);
        }

        //测试this的指向
        void test03() {
            System.out.println(this);
            System.out.println(Outer.this);
        }
    }

}

局部内部类

  • 内部类不能被public、private、static修饰；
  • 在外部类中不能创建内部类的实例；
  • 创建内部类的实例只能在包含他的方法中；
  • 内部类访问包含他的方法中的变量必须有final修饰；
  • 外部类不能访问局部内部类，只能在方法体中访问局部内部类，且访问必须在内部类定义之后。
public class Outer {

    public static void main(String[] args) {

        //测试非静态方法中的局部内部类
        Outer outer=new Outer();
        outer.test01();

        //测试静态方法中的局部内部类
        test02();

    }

    //非静态方法
    void test01(){

        class Inner{
            void testInner() {
                System.out.println("局部内部类方法");
            }
        }

        //实例化局部内部类
        Inner inner=new Inner();

        inner.testInner();

    }

    //静态方法
    static void test02() {

        class Inner{
            void testInner() {
                System.out.println("局部内部类方法");
            }
        }

        //实例化局部内部类
        Inner inner=new Inner();

        inner.testInner();

    }

}

匿名内部类

  • 没有类名，没有构造方法，但是隐式地继承一个父类或者实现一个接口。
  • 通常作为方法参数，在swing中注册监听者和启用线程中用的较多。
public class Outer {

    public static void main(String[] args) {

        //使用匿名内部类
        Thread thread=new Thread(new Runnable(){

            @Override
            public void run() {
                System.out.println("打开一个新线程");

            }

        }) ;

        //开启线程
        thread.start();
    }

}
